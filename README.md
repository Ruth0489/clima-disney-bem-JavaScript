# App Clima Disney
  
  Proyecto frontend que muestra el clima de distintas ciudades inspiradas en Disney.


---
  ## Tecnologías usadas

   - HTML
   - CSS
   - Bootstrap
   - JavaScript

   

 ## Funcionalidades

  - Visualización de ciudades en tarjetas
  - Iconos dinámicos según temperatura
  - Página de detalle con información completa
  - Diseño responsive

 Repositorio en GitHub
 


## Metodologias aplicadas
- BEM para organizacion de clases CSS
- SASS para modularizacion (variables, mixins, parciales)
- Bootstrap para layaut responsive

## Mejoras implementadas 
- Se aplicó metodología BEM para estructurar los estilos
- Se integró SASS para modularizar el CSS (variables, mixins y parciales)
- Se utilizó Bootstrap para simplificar el layout y hacerlo responsive
- Se gestionó el proyecto con Git y GitHub mediante commits descriptivos
